# 🎮 JS Quiz Game

Projeto educativo criado para praticar JavaScript de forma interativa.

## 🚀 Funcionalidades

- 25 perguntas com dificuldade progressiva
- Sistema de pontuação
- Curiosidades a cada resposta
- Bloqueio de múltiplas respostas
- Reinício do jogo

## 🧠 Conceitos aplicados

- DOM
- Eventos
- Condições (if/else)
- Arrays
- Loops
- Funções

## 🎯 Objetivo

Ajudar iniciantes a aprender JavaScript na prática, de forma divertida.

## 📦 Como usar

1. Baixe o projeto
2. Abra o index.html no navegador
3. Jogue e aprenda!
